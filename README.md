# RAPP - Social Media Application

RAPP is a modern social media application that allows users to connect, share, and interact in a dynamic online environment.

## Live Demo

Visit [https://rapp-sayuriiis-projects.vercel.app](https://rapp-sayuriiis-projects.vercel.app) to see the application in action.

## Features

- User authentication and profile management
- Post creation and sharing
- Like, comment, and bookmark functionality
- User following system
- Real-time notifications
- Responsive design for mobile and desktop

## Technology Stack

- Frontend: React.js with Next.js framework
- Backend: Next.js(actions)
- Database: PostgreSQL
- ORM: Prisma
- Authentication: [Lucia]
- Hosting: Vercel
